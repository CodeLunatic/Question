# 刷题吧小程序版

[![CodeLunatic](https://img.shields.io/badge/blog-%40CY-blue.svg)](http://cyblogs.top)

**刷题吧小程序版**是刷题吧的又一在线平台，基于`微信小程序`开发

- 同样支持`看题`，`刷题`，`错题本`，`评分`功能

- 注：`小程序版`暂已从微信下线，后台已停运，源码会在后期进行开放，后期可能会支持`广场`功能
