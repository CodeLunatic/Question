# 刷题吧小程序管理后台

[![CodeLunatic](https://img.shields.io/badge/blog-%40CY-blue.svg)](http://cyblogs.top)

**刷题吧小程序管理后台**是用来和小程序端进行交互的后台管理应用，使用SpringBoot开发

- 提供了`日常数据统计`，`用户管理`，`课程管理`，`试卷管理`，`公告管理`，`日志管理`等功能

- 注：该`后台管理程序`与`小程序后台`配套使用
